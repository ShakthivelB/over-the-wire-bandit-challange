# OverTheWire Bandit Walkthrough

![Uploading image.png…]()


A complete walkthrough of the **OverTheWire Bandit** wargame documenting each level, the Linux commands used, the concepts learned, and the solutions.

## About

Bandit is one of the best beginner-friendly Linux and cybersecurity wargames. It teaches fundamental Linux skills, networking, shell scripting, Git, file permissions, cryptography, and privilege escalation techniques that are essential for penetration testing and defensive security.

This repository contains my personal notes while solving every Bandit level.
